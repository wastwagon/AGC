# UI/UX review — human development feel (anti-“AI template”)

## What still reads “generic” without fixes

| Pattern | Why it feels AI-adjacent | Better for human dev |
|--------|---------------------------|----------------------|
| Cool gray + teal only | SaaS / consulting default | Warm neutrals + one earthy accent |
| Punchy 3-beat mission lines | Marketing generator tone | Full sentences, dignity-first language |
| Perfect 4-column stat grids | Dashboard widgets | Softer cards, human labels (“dialogues”, “people trained”) |
| Centered italic quote | Every template testimonial | Editorial pull-quote, concrete attribution |
| Heavy uppercase tracking | Corporate hero kits | Sentence case, calmer rhythm |
| Pure white cards + teal shadow | Theme forest | Paper-like surfaces, subtle borders |

## Industry references (human development / rights / think tanks)

- **Warm canvas** off-white or cream, not hospital white  
- **Copy**: outcomes for *people* (dignity, services, inclusion), not only “institutions”  
- **Layout**: slight asymmetry, breathing room, one “editorial” moment per scroll  
- **Imagery**: real faces & places when possible (content, not code)  

## Implemented in code (this pass)

- Warmer global background and body text  
- Hero overlay + copy oriented to people and partnership  
- Feature cards: softer borders, rounded-2xl, human-weight type  
- Impact block: card layout + reframed labels  
- Testimonial: pull-quote panel (warm paper tone)  
- Mission band: warm deep green, prose-style headline  
- Partner strip: earth-tone bar, conversational label
