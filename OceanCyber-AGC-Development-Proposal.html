OceanCyber is pleased to submit this proposal for the design, implementation, testing, and handover of a modern full-stack web platform for Africa Governance Centre. The solution delivered aligns with current industry practice: a TypeScript-based Next.js application, PostgreSQL data layer, authenticated administration, integrated forms and event workflows, and self-hosted production deployment (Docker Compose and related documentation for infrastructure under your control).

The total fixed professional fee is GHS 18,000.00 (exclusive of VAT, withholding tax, and other statutory levies unless agreed in writing). This document sets out scope, deliverables, timeline, and payment terms for clarity and mutual accountability.

2. Scope of work (technical)

The following reflects the full-stack development effort for the AGC platform as implemented:

2.1 Public experience

  • Multi-section marketing and information site (home, about, our work, events, news, publications, get involved, contact, applications, legal pages, and related sub-pages).
  • Responsive layout, mobile navigation patterns, and accessibility-conscious UI components.
  • On-site search and newsletter signup integration.

2.2 Custom administration (“CMS”) — no external headless CMS

  • Secure admin area with authentication.
  • Management of events, news, team, publications, programs, projects, partners, and configurable page content (including rich text where applicable).
  • Media library with upload constraints, usage checks, and administrative tooling.
  • Taxonomy configuration for content classification where applicable.
  • Review and export of inbound submissions from public forms.

2.3 Backend, data, and APIs

  • PostgreSQL schema and migrations via Prisma ORM for all content entities and submission records.
  • REST-style API routes for public forms (contact, applications, partnerships, join-us, newsletter), media, health checks, and search.
  • Optional Redis usage where configured for operational needs.

2.4 Events module (advanced)

  • Public event registration with validation, capacity and waitlist behaviour, and confirmation workflow.
  • QR-assisted registration identifiers, check-in API, staff scanning interface, and registration export.
  • Calendar (ICS) support for events.
  • Email notifications for event-related flows where outbound mail is configured on your environment (SMTP or provider API as you prefer).

3. Deliverables

  • Source code repository with build and run instructions.
  • Configured application for staging/production on client-controlled, self-hosted infrastructure (domains, TLS, and environment secrets as supplied by the client).
  • Database migrations and seed strategy as agreed.
  • Handover walkthrough covering admin usage, environment variables, and deployment.
